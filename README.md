# 🏥병원 리뷰 자동화 분석 📝

### 목적 : 수 많은 병원중에서 리뷰를 하나하나 읽어보는것이 불편하여 자동으로 리뷰를 분석하여 긍정 / 부정 비율을 계산하고 최적의 병원을 추천해주는 서비스 제공
